# Javascript ページ内のアンカーの取得

## ブラウザの「検証」でコンソールからプログラムを実行

```js
// ページ内のすべてのアンカー要素を取得
const anchors = document.querySelectorAll('a');
// アンカーのhref属性を配列に格納
const urls = Array.from(anchors).map(anchor => anchor.href);
// 結果をコンソールに表示
console.log(urls);
```
