## pandas: `notna()`:  指定カラムがnan以外のレコードのみ抽出


ダウンロードExcelファイルの `行数`カラム に値がはいっているとエラー行：

~~~py
    names = {
        "行数": "line",
        "メッセージ": "message",
    }
    df = self.dataframe(response)[["行数", "メッセージ"]].rename(columns=names)
    df = df[df["line"].notna()]
~~~
