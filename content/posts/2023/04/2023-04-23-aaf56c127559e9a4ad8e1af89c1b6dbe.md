## モジュール名からファイルを見つける


Pythonのモジュール名が与えられた時、そのモジュールの実装されているファイルのパスを調べるには、以下のようにします。

```python
import importlib.util

def get_module_path(module_name):
    spec = importlib.util.find_spec(module_name)
    if spec is None:
        return None
    return spec.origin
```

上記のように、importlib.util.find_spec()関数を使って、モジュール名からspecオブジェクトを取得し、spec.origin属性を参照することで、モジュールが実装されているファイルのパスを取得することができます。

## Djangoのアプリケーションの場合

~~~py
from django.apps import apps

class Def:
    @property
    def path(self):
        app_label = self.__module__.split(".")[0]
        conf = apps.get_app_config(app_label)
        return conf.path
~~~
