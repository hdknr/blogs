## Django: User ログイン

## ログインシグナル

Djangoでログインした時のイベントをシグナルで受け取る方法については、Djangoの公式ドキュメントに詳しく記載されています¹。以下のように、`django.contrib.auth.signals.user_logged_in`シグナルを使用して、ログイン時に実行する関数を定義し、`receiver()`デコレーターでシグナルを受信する関数を登録します。

```python
from django.contrib.auth.signals import user_logged_in
from django.dispatch import receiver

@receiver(user_logged_in)
def user_logged_in_callback(sender, request, user, **kwargs):
    # ログイン時に実行する処理
    pass
```

このように、シグナルを使用することで、ログイン時に任意の処理を実行することができます。
