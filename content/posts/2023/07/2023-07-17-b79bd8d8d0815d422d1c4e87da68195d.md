# Poetry : パッケージ依存定義 (Dependency specification)

## バージョン要件(requirements)

| 要件                                                                                                        | 例                         | 意味             |
| ----------------------------------------------------------------------------------------------------------- | -------------------------- | ---------------- |
| [Caret requirements](https://python-poetry.org/docs/dependency-specification/#caret-requirements)           | `^1.2.3`                   | `>=1.2.3 <2.0.0` |
| [Tilde requirements](https://python-poetry.org/docs/dependency-specification/#tilde-requirements)           | `~1.2.3`                   | `>=1.2.3 <1.3.0` |
| [Wildcard requirements](https://python-poetry.org/docs/dependency-specification/#wildcard-requirements)     | `1.*`                      | `>=1.0.0 <2.0.0` |
| [Inequality requirements](https://python-poetry.org/docs/dependency-specification/#inequality-requirements) | `< 2`                      | `< 2.0.0`        |
| [Exact requirements](https://python-poetry.org/docs/dependency-specification/#exact-requirements)           | `==1.2.3` もしくは `1.2.3` | `==1.2.3`        |
| [@ operator](https://python-poetry.org/docs/dependency-specification/#using-the--operator)                  |
