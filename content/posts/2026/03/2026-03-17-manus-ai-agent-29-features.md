ChatGPTに指示を出して、結果を確認して、また指示を出して——このループに疲れていないだろうか。「こっちが細かく指示しなくても、AIが全部やってくれたらいいのに」という願いは、すでに実現している。その名前が **Manus（マナス）** だ。

この記事では、Manusの全29機能を5つのカテゴリに分けて解説する。料金プランや競合との比較、業種別の活用法まで網羅する。

## Manusとは何か

### 「AIチャットボット」と「AIエージェント」の違い

まず最も重要な概念の整理から始める。

| 種別 | 代表例 | 動作 |
|------|--------|------|
| AIチャットボット | ChatGPT / Claude / Gemini | 質問→回答の1往復 |
| AIエージェント | **Manus** | タスクを丸投げ → 自動で完結 |

チャットボットは「辞書」、エージェントは「秘書」だ。Manusに「東京のAIスタートアップを30社調べて、スプレッドシートにまとめて」と指示すれば、AIが自動でブラウザを開き、検索し、情報を整理し、ファイルを作って完成品を届けてくれる。その間、別の仕事をしていてOKだ。

### 無料で試せる

最初は招待制だったが、現在は誰でも登録できる。無料プランでも毎日300クレジット分のタスクを実行可能（月1,500上限あり）。

### 実力は？

GAIAベンチマーク（実際のタスクをどれだけ正確にこなせるかを測るテスト）において、ManusはOpenAIのDeep Researchに匹敵する高スコアを記録したと複数のメディアが報じている。「実務能力で世界トップクラス」という評価は多くのメディアで一致している。

## 第1章：Manusの全29機能

Manusには5つのカテゴリ、計29の機能がある。

### コア機能（日常的に使う基本機能）

#### ① 自律タスク実行（Autonomous Task Execution）

Manusの心臓部。指示を出すだけで、AIが計画から納品まで全自動でやる。

タスクを入力すると、Manusはまず計画を立てる。「何を調べるか」「どのツールを使うか」「どの順番でやるか」を自分で判断し、ブラウザで検索したりコードを書いて実行したりしながら完了させる。全作業はクラウド上のサンドボックスで行われるため、PCには影響しない。画面を閉じても作業は続き、完了したら通知が届く。

実行例：
- 「この契約書PDFの要点を抽出して、リスク項目をハイライトして」
- 「A社の過去3年の決算資料を分析して、成長率と利益率の推移をグラフにして」
- 「今月の営業チームの活動データをまとめて、週次ミーティング用の資料を作って」

#### ② Wide Research（ワイドリサーチ）

100体以上のAIを同時に走らせて、大規模リサーチを一気にやる。

通常のAIは1つのタスクを1つのエージェントが処理するが、Wide Researchはタスクを細かく分解して100体以上のAIエージェントを同時並行で走らせる。1社調べるのも100社調べるのも、かかる時間はほぼ同じだ。

OpenAI Deep Researchとの違い：

| 機能 | 特徴 |
|------|------|
| Deep Research | 1つのテーマを深く掘り下げる（深さ重視） |
| Wide Research | 多数のテーマを同時に広く調べる（広さ重視） |

注意点：クレジット消費が大きいため、まず10〜20社でテストしてからスケールアップするのが鉄則。最大250件まで対応。

#### ③ Browser Operator（ブラウザオペレーター）

ChromeまたはEdgeに拡張機能をインストールすると、Manusがブラウザを直接操作できるようになる。最大のポイントは、既存のログイン情報がそのまま使えること。つまり、ログインが必要な社内ツール、CRM、管理画面も操作可能だ。

主な活用例：
1. CTO 50人の名前・会社名・プロフィールURLを収集
2. ECサイトで特定商品の価格を毎日モニタリング
3. Wantedlyで求人情報を一括収集
4. 競合5社のX（Twitter）アカウント分析
5. Googleフォームへのデータ一括入力

セキュリティ面：毎回の操作で許可が必要で、全操作ログが記録される。パスワードは保存されない。

#### ④ AI Slides（AIスライド）

情報収集→構成作成→デザイン→スピーカーノートを全自動で、約5〜10分でスライドを作成する。

「Nano Banana Pro」モデルを搭載し、スタジオクオリティの画像がスライド内に自動生成される。生成後はポイント＆クリックで部分編集も可能。

#### ⑤ Mail Manus（メールマナス）

メールを受信するだけで、AIが自動でタスクを実行する。

登録すると専用ボットアドレス（例：`your-name-abc123@mail.manus.ai`）が発行される。このアドレスにメールを転送するだけで、Manusが自動でタスクを開始する。

活用例：
- 旅行予約の確認メールを転送 → 旅程表を自動作成
- 経費の領収書メールを転送 → 経費レポートに自動整理
- 英語メールを転送 → 日本語で要約して返信の下書きを作成

#### ⑥ Chat Mode（チャットモード）

通常の会話モード。ChatGPTのように使えて、クレジット消費ゼロ。

Agent Mode（タスク自動実行）とChat Mode（通常会話）を使い分けるのがクレジット節約の最大のコツ。迷ったらChat Modeで相談してから、必要に応じてAgent Modeに切り替える。

#### ⑦ Task Replay（タスクリプレイ）

AIがやった作業の全ステップを動画のように再生できる。どのサイトを開いたか、何を検索したか、どんな判断をしたか、どのファイルを作ったか——全てのステップが記録されている。

ビジネスで「なぜこの結果になったのか」を上司やクライアントに説明する際に役立つ透明性機能だ。

#### ⑧ Schedule Task（スケジュールタスク）

「毎朝9時」「毎週月曜」「毎月1日」のように定期的なタスクを自動で繰り返し実行してくれる。一回設定すれば、あとはずっと自動だ。

設定例：
1. 毎朝のニュースダイジェスト：業界ニュースを5本要約 → Notionに追加
2. 週次競合モニタリング：競合5社のプレスリリース・SNSをチェック → スプレッドシートに記録
3. 月次KPIレポート生成：データ収集 → グラフ付きPDFレポートを自動作成
4. 毎日のSNSトレンド分析：X（Twitter）の業界トレンドを抽出 → トップ10まとめ
5. 月次の請求書処理：未処理の請求書メールを検索 → リスト化して通知

後述のConnectorsと組み合わせると真価を発揮する。

### 生成・開発機能（ものを作る5機能）

#### ⑨ Website / App Builder（ウェブサイト・アプリビルダー）

プログラミング知識ゼロで、言葉で説明するだけでWebサイト・Webアプリを自動生成する。

作れるもの：
- SaaSダッシュボード（管理画面、グラフ、ユーザー管理機能付き）
- ECサイト（商品カタログ、カート、Stripe決済、在庫管理）
- ランディングページ（CTA配置、コンバージョン最適化済み）
- 予約システム（カレンダー連携、自動確認メール）
- 会員制サイト（ログイン認証、コンテンツ制限）
- SEOブログ（スキーママークアップ自動生成）

Figmaインポート、カスタムドメイン、Stripe決済統合、コード完全アクセスに対応。アイデアからデプロイまで数分で完了する。

#### ⑩ Mobile Development（モバイル開発）

iOS/Androidアプリを言葉だけで開発。2025年12月のManus 1.6で追加された。「こんなアプリが欲しい」と説明するだけで、エンドツーエンドで開発してくれる。スマホでリアルタイムプレビューも可能。

注意：最終的なストア公開は自分で管理する必要がある。

#### ⑪ Design View（デザインビュー）

AIが生成した画像をキャンバス上で直接編集できる。2025年12月のManus 1.6で追加。マークツールで特定要素を選択してテキスト指示で精密編集でき、参照画像のアップロードにも対応する。

#### ⑫ Image Generator（画像生成）

SNS投稿用グラフィック、バナー、イラスト、プレゼン用の図解をテキストから生成する。スライド作成やWebサイト構築のワークフロー内で画像を同時生成できるため、別のツールに切り替える必要がない。

#### ⑬ Data Visualization（データ可視化）

CSVやスプレッドシートのデータをグラフ・チャート・ダッシュボードに自動変換する。

対応入力：CSV、Excel  
出力形式：スライドデック、対話型ダッシュボード、詳細レポート、Webページ  
対応チャート：棒グラフ、折れ線グラフ、円グラフ、散布図、ヒートマップ、レーダーチャート（最大5種類同時）

### 連携・拡張機能

#### ⑭ Connectors（コネクター）

Gmail、Slack、Notion、Google Drive、Google Calendar、HubSpot等の主要サービスとManusを連携させる。1つのプロンプトで複数アプリを横断操作できる。

設定例（Gmail）：Settings → Connectors → Gmail → Connect → OAuth認証。これだけで完了。

活用例：「Gmailからクライアントのフィードバックを検索し、Notionのデータベースに追記し、要約をSlackに投稿して」→ 3つのツールを手動で行き来する作業がManus1つで完結。

#### ⑮ Manus Agents（マナスエージェント）

Telegramからタスクを依頼できる機能。2026年2月16日発表。QRコード読み込みで約1分でセットアップ完了。

できること：テキスト・音声・画像・ドキュメントでタスクの実行、モデルの選択（Max / Lite）。移動中にスマホからサクッとタスクを投げたい時に便利だ。

#### ⑯ API

開発者が自分のアプリからManusの機能を呼び出せる。Slackボット、Webアプリ、社内ツールにManusの自律実行能力を組み込める。技術者向け機能。

#### ⑰ Projects & Library（プロジェクト＆ライブラリ）

タスクをプロジェクト単位で整理し、過去の成果物を再利用できる。プロジェクトに「マスター指示」「Knowledge Base」「Connectors」「Skills」を設定しておけば、新規タスク作成時に全設定が自動適用される。

#### ⑱ Collaboration / Collab（コラボレーション）

チームメンバーと同じタスク上で共同作業できる。クレジット消費はタスクオーナーのみで、招待された人は無料。プロジェクトを複数人で進める時に最適だ。

#### ⑲ Manus Max（マナスマックス）

2025年12月のManus 1.6で登場したフラッグシップモデル。高度な計画能力と問題解決アーキテクチャを搭載し、タスクの一発成功率が大幅に向上している。スプレッドシートの高度な処理（財務モデリング、データ分析）や Web開発が得意。

Telegram経由ではMaxとLiteを選択可能。複雑なタスクにはMax、簡単な処理にはLiteと使い分けられる。

#### ⑳ MCP Servers

社内ツールや独自APIとManusをつなぐ技術者向け連携機能。MCP（Model Context Protocol）はAnthropicが策定したオープン標準。技術者向けのため、一般ユーザーは気にしなくてOK。

### 2026年の新機能

#### ㉑ Meeting Minutes（ミーティングミニッツ）

対面会議のリアルタイム議事録を自動作成する。2026年1月12日発表。

手順：
1. Meeting Recordingアイコンをクリック
2. 会議を録音
3. Finishボタンで終了
4. AIが自動で要約・構造化

スピーカー認識（誰が何を言ったか自動判別）、アクションアイテムの自動割り当て、インターネット接続が切れても記録継続といった特徴がある。

注意：対面会議専用。Zoom等のオンライン会議には非対応。録音自体は無料だが、分析・要約はクレジット消費あり。

#### ㉒ Agent Skills（エージェントスキル）

AIの「得意技」をパッケージ化して再利用できる。2026年1月27日発表、Anthropicのオープン標準を採用。

うまくいったワークフローを「スキル」として保存し、スラッシュコマンド（`/スキル名`）で発動できる。GitHubからインポートも可能。Team Skill Libraryで組織全体の知識を共有できる。

#### ㉓ Project Skills（プロジェクトスキル）

プロジェクト専用のスキルライブラリを構築できる。2026年2月12日発表。プロジェクトごとに使用するスキルを設定し、ロック機能で「ゴールドスタンダード」ワークフローを確立できる。新メンバーのオンボーディング加速にも有効だ。

#### ㉔ Data Sources（データソース）

Similarweb等のプレミアムデータに自動アクセスできる。2026年1月13日にSimilarwebとの公式パートナーシップが発表された。APIキー不要で、12ヶ月の履歴データ、トラフィック＆エンゲージメント、ランキング、地理的データにアクセスできる。市場調査・投資分析・競合分析に強力だ。

注意：Similarwebデータの利用はクレジット消費あり。

#### ㉕ Multimedia Processing（マルチメディア処理）

画像生成・画像理解・動画理解・音声合成（TTS）・音声テキスト化（STT）に対応する。Telegram経由で音声メッセージを送ると自動転写されるのもこの機能のおかげだ。

### インフラ機能（裏側で動いている技術基盤）

#### ㉖ Sandbox / Cloud Computer（サンドボックス）

タスクごとに完全に隔離されたUbuntu仮想マシンがクラウド上で動いている。ソフトウェアのインストールやカスタムツールの作成が可能で、並列実行に対応している。非アクティブ時は自動スリープし、一定期間後に自動削除される。

#### ㉗ Cloud Browser（クラウドブラウザ）

Browser Operator（③）はローカルブラウザを使うが、Cloud Browserはクラウド上の隔離ブラウザだ。AIが認証チェックやCAPTCHAに遭遇した場合、ユーザーが一時的に介入して手動で解決し、その後AIに操作を戻せる「テイクオーバー機能」が特徴的だ。

#### ㉘ Microsoft 365統合

2025年11月発表。MicrosoftのAgent 365プラットフォームにManusが統合されている。企業のIT管理者向けのエージェント管理、リスクベースのアクセス制御、セキュリティ対策を含む。主に大企業向けの機能だ。

#### ㉙ Manus Academy（マナスアカデミー）

Manusの使い方を学べる公式学習プラットフォーム。Build Clubとの提携で運営されており、30時間以上の構造化学習コンテンツ、20以上の実践的ユースケースを提供する。最初の500人の修了者に8,000 Manusクレジット（約$40相当）が付与される。

## 知っておくべき技術的な裏側

### Manusが使っているAIモデル

Manusは1つのAIモデルではなく、複数のAIモデルを組み合わせて動いている。Anthropic Claude（Sonnet系）がメインの思考エンジンと報じられており、Google Cloud Platformも活用している。

「ManusはClaudeを使ってるの？」——答えはYes（一部）。しかしManusの価値はAIモデル単体ではなく、「複数のAIモデル + ブラウザ操作 + コード実行 + ファイル管理 + ツール連携」を組み合わせたオーケストレーション（統合制御）にある。個々のパーツは他のAIでもできること。計画から納品まで全自動でつなげるのがManusの独自性だ。

### 29種類のツールを内蔵

Manusは内部に29種類のツール（Webブラウジング、コーディング、データ分析、メディア作成など）を持ち、どのツールをどの順番で使うかをAIが自分で判断する。人間のプロジェクトマネージャーが「この作業はデザイナーに、この作業はエンジニアに」と振り分けるように、ManusのAIがツールを振り分けているイメージだ。

## 第2章：業種別・Manusの使い方ガイド

### マーケターの場合

**リサーチ系：**
- Wide Researchで競合100社の広告戦略を一括分析
- Data Sources + Similarweb連携で競合サイトのトラフィックデータを自動取得

**コンテンツ制作系：**
- AI Slidesで毎月のマーケティングレポートを自動生成
- Website Builderでキャンペーン用ランディングページを数分で作成
- Image Generatorでバナー・SNS投稿用画像を量産

**自動化ワークフロー例（月次マーケティングレポート）：**
1. Schedule Taskで毎月1日に実行設定
2. Google Analytics Connectorからデータ取得
3. Data Visualizationでグラフ・チャート生成
4. AI Slidesでプレゼン資料に自動変換
5. 完成したレポートをSlackのチャンネルに自動投稿

手動でやると半日かかる作業が、放置で完了する。

### 営業・セールスの場合

**リード獲得：**
- Browser OperatorでLinkedInから見込み客情報を自動収集
- Wide Researchで「○○業界の意思決定者100人のプロフィールをまとめて」
- 収集データをNotionのCRMに自動投入

**提案書作成：**
- AI Slidesで「A社向けの提案書を作って。課題は○○、提案は○○」
- Data Visualizationで売上予測シミュレーショングラフを3パターン生成
- 競合製品との比較表をWide Researchで自動生成

### エンジニア・開発者の場合

**開発系：**
- Website / App BuilderでMVP（最小限の製品）を爆速で作成
- Mobile Developmentでアプリのプロトタイプを数時間で構築
- APIで自社ツールにManusの機能を組み込み

**自動化系：**
- Schedule Taskで「毎朝、依存パッケージのセキュリティアラートをチェックしてSlackに通知」
- MCP Serversで社内の独自APIとManusを連携
- Agent Skillsでデプロイ手順をスキル化して再利用

## まとめ

Manusは「指示→回答」のチャットボットから「指示→全自動完了」のエージェントへと、AIとの関わり方を根本から変えるツールだ。

全29機能を一覧すると：
- **コア機能**（①〜⑧）：自律実行、Wide Research、ブラウザ操作、スライド生成、メール自動化など
- **生成・開発**（⑨〜⑬）：Website Builder、モバイル開発、画像生成、データ可視化
- **連携・拡張**（⑭〜⑳）：Connectors、Telegram、API、プロジェクト管理、MCP
- **2026年の新機能**（㉑〜㉕）：議事録自動化、スキルの再利用、プレミアムデータ連携
- **インフラ**（㉖〜㉙）：サンドボックス、クラウドブラウザ、Microsoft 365統合、学習プラットフォーム

無料プランで毎日300クレジット分のタスクを試せる。まずはChat Modeで相談してから、Agent Modeで実際のタスクを体験してみるのが一番の近道だ。
