Claude Code を使いこなすうえで押さえておきたいコマンド・指示パターンを10個まとめました。
これを知っているか知らないかで、作業スピードに大きな差が生まれます。

## コマンド一覧

### 1. `/edit` — 部分修正で爆速

ファイル全体を書き直すのではなく、特定の箇所だけを修正したいときに使います。

```
/edit auth.py の login 関数のバリデーション処理を修正して
```

変更箇所を絞ることで、Claude が意図を正確に把握しやすくなり、意図しない書き換えを防げます。

### 2. `/diff` — 変更点だけ確認

Claude が加えた変更内容を一覧表示します。大量に編集した後に「何が変わったか」を素早く把握できます。

```
/diff
```

`git diff` の感覚で使えるため、コードレビュー前の最終確認に重宝します。

### 3. `/undo` — 即戻せる

直前の変更を取り消します。「やっぱり前の方が良かった」というときに即座に元に戻せます。

```
/undo
```

Claude Code はセッション内の変更履歴を保持しているため、複数回の `/undo` で遡ることも可能です。

### 4. `/run` — 実行まで自動

コードの生成だけでなく、その場で実行まで完結させます。スクリプトの動作確認やテスト実行を素早く回せます。

```
/run scripts/migrate.py を実行して結果を確認して
```

実行結果のエラーも自動で拾い、修正まで一気に進めてくれます。

### 5. `/test` — テスト生成

実装コードに対応するユニットテストを自動生成します。テストを書く手間を大幅に削減できます。

```
/test UserService クラスのテストを生成して
```

既存のテストフレームワーク（pytest、Jest など）を検出して、プロジェクトのスタイルに合わせたコードを出力してくれます。

### 6. `/fix` — エラー修正

エラーメッセージをそのまま貼り付けるだけで、Claude が原因を分析して修正します。

```
/fix TypeError: 'NoneType' object is not subscriptable
```

スタックトレースごと渡すと、より正確な修正が得られます。

### 7. `/refactor` — 最適化

動いているコードをより読みやすく・効率的な構造に書き直します。

```
/refactor この関数を読みやすくリファクタリングして
```

「どう直すか」を指示しなくても、Claude がベストプラクティスに沿って提案してくれます。

### 8. `/search` — 全体検索

プロジェクト全体から特定のキーワードや処理を含むファイルを探します。

```
/search JWT トークンの検証処理を行っている箇所を全部探して
```

`grep` や IDE の検索より自然言語で指示できるのが利点です。

### 9. `/commit` — コミットメッセージ作成

変更内容を解析して、適切なコミットメッセージを自動生成します。

```
/commit
```

Conventional Commits 形式（`feat:`, `fix:`, `refactor:` など）に合わせたメッセージを提案してくれます。

### 10. `/doc` — ドキュメント化

関数やクラスに docstring・コメントを自動で追加します。API ドキュメントの生成にも活用できます。

```
/doc payment.py の関数に docstring を追加して
```

## 効果的な使い方のコツ

**具体的に伝える**  
「かっこよくして」より「黒背景・ゴールドのグラデーション・グロー効果を追加して」のように詳細に指示するほど、期待通りの出力が得られます。

**一度に一つ**  
複数の変更を一気に頼むより、一つずつ確認しながら進める方が品質が高まります。

**エラーはそのまま貼る**  
エラーメッセージをコピーして Claude Code に渡すと、自動で原因を特定して修正してくれます。

**参考サイトのイメージを伝える**  
「〇〇のサイトみたいに」「Xのプロフィール画像みたいに丸く」など、イメージを共有することで Claude が意図を正確に解釈してくれます。

## まとめ

Claude Code のコマンドを活用することで、コーディングの各フェーズ（実装→テスト→修正→コミット→ドキュメント）を効率的に進められます。
「何をしたいか」を明確に伝えることが、AI との協働を最大化する鍵です。
