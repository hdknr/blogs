## 概要

AI を活用して動画広告を自動生成するツールカテゴリ。特に UGC（User Generated Content）スタイル — 一般ユーザーが撮影したような自然な見た目の動画 — を AI アバターで生成する分野が急成長している。SNS 広告のクリエイティブ制作コストを大幅に削減し、大量のA/Bテストを可能にする。

## 主要ツール比較

| ツール | 強み | 最適なユースケース |
|--------|------|------------------|
| **Arcads** | UGC リアルさが最高水準、バッチ生成 | SNS 有料広告の大量クリエイティブテスト |
| **Creatify** | URL → 動画の完全自動化 | EC の SKU 別動画量産 |
| **HeyGen** | 多言語（175言語）、コスパ良 | 企業向け多言語コンテンツ |
| **Synthesia** | エンタープライズ実績、安定性 | 大企業の研修・教育コンテンツ |

## ユースケース別の選択指針

- **SNS 有料広告（Meta / TikTok）のクリエイティブテスト** → Arcads（UGC 品質と大量テスト）
- **EC商品の動画量産** → Creatify（URL から自動生成）
- **企業向け多言語コンテンツ** → HeyGen（175言語対応 + リップシンク翻訳）
- **エンタープライズ研修・教育** → Synthesia（LMS 連携、セキュリティ対応）

## 料金感の目安（2026年4月時点）

| ツール | 最安プラン | 備考 |
|--------|-----------|------|
| Arcads | $110/月（10本） | 無料トライアルなし |
| Creatify | $39/月（100クレジット） | 無料枠あり |
| HeyGen | $24/月（Creator：無制限） | 無料枠あり（透かし付き） |
| Synthesia | $18/月（Starter：年120分） | 無料枠あり |

## API 連携の可否

プログラムからの自動連携を前提とする場合:
- **Creatify**: 全プランで API 利用可
- **HeyGen**: 従量課金（1分=1クレジット）
- **Arcads**: Pro プラン（カスタム）限定
- **Synthesia**: Enterprise 限定
