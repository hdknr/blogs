## 概要

PostgreSQL を基盤とした BaaS。PostgREST でスキーマから自動 REST API 生成、PostGraphile で GraphQL 対応。AWS 上でのセルフホスト構成も Terraform で可能。Firebase ライクなコンセプトで PostgreSQL の柔軟性を備える。
