## 概要

Arcads（arcads.ai）は、テキストの広告台本から UGC（ユーザー生成コンテンツ）スタイルの動画広告を AI で自動生成するプラットフォーム。2024年にフランスで創業し、18ヶ月で売上1,300万ドルを達成。シードラウンドで1,600万ドルを調達（Eurazeo リード）。

## 主な特徴

- **1,000以上のAIアクター**: 自然な表情・ジェスチャー・口の動きを再現。性別・年齢・人種・背景でフィルタリング可能
- **バッチ生成**: 1台本からアクター × フック × CTA のバリエーションを一括生成。大規模A/Bテストが低コストで実現
- **感情コントロール**: テキスト指示で「驚き」「興奮」などの感情表現を制御
- **30以上の言語対応**: 多言語広告の量産に対応
- **API連携**: Pro プラン限定（40 req/s、clientId/clientSecret 認証）

## 料金プラン

| プラン | 月額 | 動画数 | 1本単価 |
|--------|------|--------|--------|
| Starter | $110 | 10本 | 約$11 |
| Creator | $220 | 20本 | 約$11 |
| Pro | カスタム | 大量 + API | 要問合せ |

- 無料トライアルなし、年払いで15〜20%割引

## 適したユースケース

- SNS 有料広告（Meta / TikTok）のクリエイティブ大量テスト
- デジタル商品（SaaS・アプリ・オンラインコース）の体験談風広告
- OpenClaw などのエージェントと API 連携した広告運用自動化

## 競合ツールとの比較

| ツール | 強み | 向いているユースケース |
|--------|------|----------------------|
| **Arcads** | UGC リアルさが最高水準 | SNS 有料広告の大量テスト |
| **Creatify** | URL → 動画の自動化 | EC の SKU 別動画量産 |
| **HeyGen** | 多言語・企業向け | 研修・プロダクトデモ |
| **Synthesia** | エンタープライズ実績 | 大企業研修・教育 |
