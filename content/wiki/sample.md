## Hugo + GitHub Pages セットアップ

### 前提条件

- Go がインストール済み
- GitHub アカウント

### 手順

1. Hugo をインストール
2. `hugo new site` でサイト作成
3. テーマを追加（PaperMod）
4. `hugo.toml` を設定
5. GitHub Actions でデプロイ設定
